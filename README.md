# CDFlow

轻量级冷却管理器美化插件，专注简洁与实用。  
基于魔兽世界 12.0 内置的 **Cooldown Manager** 系统，支持图标样式、布局、高亮特效等深度自定义。

---

## 功能概览

- **图标样式**：尺寸、裁剪、边框、Debuff 红框屏蔽
- **布局控制**：增长方向（向下/向上）、间距、每行数量、行尺寸覆盖、追踪状态栏增长方向
- **文字叠层**：堆叠数字样式、快捷键显示（自动识别 + 手动覆盖）、冷却读秒样式
- **高亮特效**：技能激活高亮 + Buff 增益高亮（支持技能 ID 过滤，多种特效可选）
- **监控条**：自定义 Buff 堆叠条 / 技能充能条 / 技能冷却条，支持专精显示、从 CDM 隐藏
- **配置管理**：多配置文件、专精自动切换、配置导入导出

## 安装

1. 解压到：
   ```
   World of Warcraft\_retail_\Interface\AddOns\CDFlow
   ```
2. 进入游戏后使用 `/cdf` 打开设置。

## 依赖

- 魔兽世界：12.0+（Interface 120000+）
- 内置库：LibStub、AceGUI-3.0、LibCustomGlow-1.0、CallbackHandler-1.0
